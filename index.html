Secure Document Verification using Blockchain

Upload health insurance documents, generate SHA-256 hashes, store them as blockchain blocks, and verify document integrity anytime.

📄 Upload Documents

Upload Proposal Forms, Medical Reports, ECG, KYC, Lab Reports and Policy Documents.

🔐 SHA-256 Hashing

Every uploaded file receives a unique cryptographic fingerprint.

⛓ Blockchain Storage

Each document becomes a blockchain block linked with the previous block.

✔ Tamper Detection

Verify whether a document has been modified after upload.

How It Works

  1. Upload a Health Insurance Document.
  2. Generate SHA-256 Hash.
  3. Create a Blockchain Block.
  4. Store the Block in Local Storage.
  5. Verify the Document Later.
